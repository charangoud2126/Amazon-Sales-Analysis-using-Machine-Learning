# 🛒 Amazon Sales Analysis using Machine Learning

## Project Overview

This project focuses on **analyzing and predicting Amazon product sales using Machine Learning techniques**. The goal is to extract meaningful insights from sales data, understand key factors influencing product performance, and build predictive models to support data-driven business decisions.

Through **Exploratory Data Analysis (EDA)** and **Machine Learning algorithms**, this project demonstrates how businesses can optimize pricing, improve product strategies, and forecast sales trends.

---

## Objectives

* Perform **data cleaning and preprocessing** on Amazon sales data
* Conduct **Exploratory Data Analysis (EDA)** to identify sales patterns
* Visualize key insights using data visualization tools
* Build **Machine Learning models** to analyze and predict sales trends
* Evaluate model performance using appropriate metrics

---

## Dataset

The dataset contains information related to Amazon product sales such as:

* Product Category
* Product Ratings
* Discount Percentage
* Price Information
* Number of Reviews
* Sales Performance Indicators

Dataset file used in this project:

```
amazon_sales_dataset.csv
```

## Project Workflow

### 1️⃣ Data Collection

Load the Amazon sales dataset using Pandas.

### 2️⃣ Data Preprocessing

* Handling missing values
* Data type conversion
* Feature selection

### 3️⃣ Exploratory Data Analysis (EDA)

* Sales distribution analysis
* Rating vs sales insights
* Discount impact analysis
* Category performance analysis

### 4️⃣ Data Visualization

Visual insights using:

* Bar charts
* Correlation heatmaps
* Distribution plots
* Trend analysis

### 5️⃣ Machine Learning Model

* Feature engineering
* Train/test split
* Model training using **Scikit-learn**

### 6️⃣ Model Evaluation

Evaluate performance using metrics such as:

* Accuracy
* Mean Squared Error (MSE)
* R² Score

---

## Key Insights

Some insights discovered during analysis:

* Discount percentage strongly influences product sales.
* Highly rated products receive significantly more purchases.
* Certain product categories dominate overall revenue.
* Review count is a strong indicator of product popularity.
